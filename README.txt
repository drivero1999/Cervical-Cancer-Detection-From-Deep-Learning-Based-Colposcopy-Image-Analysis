En las carpetas Two class classification y Three class classification, se encuentro las arquitecturas
usadas para la clasificacion de imagenes de cuello uterino segun su grado de riesgo/displasia. 
Ademas de encontrar los resultados previamente cargados de las ejecuciones de cada uno junto a sus metricas.

En la carpeta Architectures encontrada el esquema de las redes que fueron implementadas, pudiendo consultar el
numero de capas y neuronas usadas en cada caso, y como fueron conectadas entre si.

Finalmente, en la carpeta Android Studio code encontrada un archivo .txt, en el cual encontrara un link donde podra
descargar la interfaz que fue implementada, la cual le permitira tomar una foto de un cuello uterino o cargarla desde
la galeria de su dispositivo Android, para darle inmediatamente un diagnostico.
